Welcome to the ARF's Obsidian Geochemistry Visualization App!

This R-Shiny App plots artifact geochemistry values on a biplot against ellipses representing geochemical values from well-characterized obsidian sources.

Source geochemical data is derived from Neutron Activation Analysis studies conducted by the Missouri University Research Reactor and provided by Mike Glascock for hosting on this OpenContext Project. The geographical span of the source data is currently limited to the continent of South America.

Intended workflow

  1. Select obsidian sources in your study region by Country and Source Names.
  2. Upload artifact data CSV (Comma Separated Values table) under "Data Management". See this file for an example of the expect CSV format derived from a calibrated X-Ray Fluorescence instrument.
  3. Choose source chemical elements that provide good discrimination between sources in your region. Because these are based on NAA the most promising elements are Rb, Th, and sometimes Sr (when > 50ppm) and Zr when U is low.
  4. Assign samples to sources by typing source name into the Notes field. Click Download to save the modified table to a local CSV file. [Note: the Download feature is not currently working. Workaround: do this step separately]

For more details on obsidian source chemistry or to download the NAA data directly browse the project records on the OpenContext repository.

Caveats

This workflow involves overlaying XRF artifact data on NAA derived chemical values and there are numerous complications in this comparison with which you should be familiar.
